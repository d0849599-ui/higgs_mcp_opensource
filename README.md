### ⚛️ Overview
A bridge between Monte Carlo physics simulations and generative AI visualization.

### 🛠 Tech Stack
- **Simulation:** Python 3.14 + NumPy + Matplotlib
- **Local AI:** Ollama (Gemma4 / Llama2)
- **Cloud Rendering:** Higgsfield AI (Nano Banana Pro)
- **Architecture:** Model Context Protocol (MCP)

### 📊 Scientific Logic
The core simulation calculates particle interaction with the Higgs Field (v = 246 GeV) to determine rest masses across four particle generations.
